domManipulationService
service in module poBox

Description

Style service to run remaining formatting when Angular templates are loaded. We need this service mainly to update html tags outside of our Angular application that we can't directly theme by using Angular directives.

Methods

  • scrollToTop()

    Scrolls the user to the top of the page (for when page navigation occurs)

  • updateCMSBreadcrumb(pageName)

    Set / Update the CMS breadcrumb according to the current Angular template loaded.

    Parameters
    ParamTypeDetails
    pageNameString

    The page Name used to update the breadcrumb.
