NPM and NODE

Make sure you have NPM (version 3+) installed on your development environment.

To check your npm version :

$ npm -v

If you need to upgrade your npm version, try (use sudo if needed):

$ npm cache clean -f
$ npm install -g npm

# Or something like that if you need to specify a proxy (e.g if you are using the ADC network) :
$ npm --https-proxy http://10.23.12.100:8080 install -g npm

Make sure you have NODE (version 6+) installed on your development environment.

To check your npm version :

$ node -v

If you need to upgrade your node version, you can download and use the tool nvm :

# Download and install nvm - More info : https://github.com/creationix/nvm
$ wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.31.2/install.sh | bash
$ source ~/.nvm/nvm.sh

# List all the node versions available
$ nvm ls-remote

# Install the latest version of node
$ nvm install 6.3.0

# Check the node version
$ node -v

# If you get N/A when running ls-remote you export the following mirror and retry :
$ export NVM_NODEJS_ORG_MIRROR=http://nodejs.org/dist

Install npm dependencies (for development purpose only) by running the following command inside the rml_pobox/source folder:

$ npm install

Gulp Task Commands

Run all the gulp tasks defined:

$ gulp

Run a gulp build:

$ gulp build

Run a gulp build without tests:

$ gulp quickbuild

Re-generate the Angular Documentation:

$ gulp doc

Re-generate the sprite.png file:

$ gulp sprite

Sort package.json Dependencies and DevDependencies

$ gulp sort

Run the Angular Unit Tests:

$ gulp test

Run the Behaviour-Driven Development (BDD) Tests:

$ gulp e2e-test

Run the Gulp Linter:

$ gulp lint

Documentation

Once $ gulp doc has run, the AngularJS documentation can be accessed at the following URL: http://wwwvagrant.royalmail.com/sites/all/modules/custom/rml_pobox/source/doc/index.html
